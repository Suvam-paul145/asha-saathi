### 1. Fork the Repository
Click the **Fork** button on GitHub and clone your fork locally.

- git clone https://github.com/AnushSingla/asha-saathi.git
- cd asha-saath


---

### 2. Create a New Branch
Always create a new branch for your work.

- git checkout -b feature/your-feature-name


Examples:
- `feature/video-upload`
- `fix/auth-bug`
- `docs/readme-update`

---

### 3. Set Up the Project Locally

Follow the setup instructions in the `README.md` to run both frontend and backend locally.

Make sure:
- Backend runs without errors
- Frontend loads correctly
- Environment variables are configured

---

### 4. Make Your Changes

Guidelines:
- Keep code clean and readable
- Follow existing project structure
- Use meaningful variable and function names
- Write reusable and modular code

---

### 5. Commit Your Changes

Write clear and descriptive commit messages.

git commit -m "Add video like feature"


---

### 6. Push and Create a Pull Request

git push origin feature/your-feature-name


Then open a **Pull Request** on GitHub:
- Clearly describe what you changed
- Link related issues (if any)
- Add screenshots for UI changes
